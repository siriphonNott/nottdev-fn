# Nottdev Function
function utils for javscript lover

## Features
- uniqueArr

# Installing
#### Package manager
Using npm or yarn or pnpm:
```sh
npm install nottdev-fn
or
yarn add nottdev-fn
or
pnpm add nottdev-fn
```
## Using
```sh
import { uniqueArr } from 'nottdev-fn'
let myNums = [1, 2, 3, 1, 4, 1, 2, 5, 3, 4];
let uniqueNums = uniqueArr(myNums);
console.log(uniqueNums);
```
